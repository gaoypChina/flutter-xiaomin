## 目录

```
components --------------------------------- 公共组件
    button --------------------------------- 按钮组件
    choose_file ---------------------------- 选择文件
    input ---------------------------------- 输入框
    tag ------------------------------------ 标签
    user_avatar ---------------------------- 用户头像
    verification_code_button --------------- 验证码倒计时按钮
    
layout ------------------------------------- 全局布局
pages -------------------------------------- 页面
    bill ----------------------------------- 账单
        add_bill --------------------------- 新增账单
        
    forget_password ------------------------ 忘记密码
    home ----------------------------------- 入口页
    login ---------------------------------- 登录
    notepad -------------------------------- 记事本
        add_note --------------------------- 添加事件
        components ------------------------- 记事本页面组件
        note_detail ------------------------ 事件详情
        note_drafts ------------------------ 草稿
        
    preview_file --------------------------- 文件预览(支持图片和视频预览)
    record --------------------------------- 经期
    register ------------------------------- 注册
    splash_screen -------------------------- 闪屏页
    target --------------------------------- 目标
        choose_target ---------------------- 选择目标
        components ------------------------- 目标组件
        create_target ---------------------- 新建目标
        
    user_info ------------------------------ 用户信息
        
utils -------------------------------------- 工具
main.dart ---------------------------------- 项目入口
```
